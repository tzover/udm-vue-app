# テンプレート構文

## 種類 と 役割

| 種類    | 役割                                                           |
| :------ | :------------------------------------------------------------- |
| v-once  | 初回だけ評価し、以降は静的コンテンツとして扱う                 |
| v-pre   | 要素とその子要素の評価をスキップ                               |
| v-html  | HTML を挿入                                                    |
| v-cloak | ページ読み込み完了時に内容を表示する（読み込み中は表示しない） |
| v-text  | {{}} の代わりに使うもの                                        |
