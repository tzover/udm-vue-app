# Vue.js 3

# Node version

- node v16.13.0
- npm 8.1.0
- yarn 1.22.15

# Create Vue environment

## install cli

```
yarn add @vue/cli
```

> "@vue/cli": "^5.0.4"

## Create vue env

```
vue create app
```

- Please pick a preset: Default ([Vue 3] babel, eslint)
- Pick the package manager to use when installing dependencies: Yarn

```
cd app
yarn serve -- --port 3000
```
